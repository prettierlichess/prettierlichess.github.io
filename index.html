Lichess is a great platform that provides lots of fantastic free services like free analysis, studies, tournaments, and more. However, the site doesn't feel very contemporary, and this can turn many users off towards other platforms, and ultimately off an incredibly useful and amazing resource.

Prettier Lichess is an attempt to fix that.
Download it on the Chrome Web Store!
Disclaimer: This extension is still in active development, and as such parts of the site are not yet completely styled. The homepage, profile page, match page, puzzle page, tournament page, and various study pages are all currently styled. In the roadmap, the entire site is set to be styled, as well as more custom color choices being available, and possibly user picked colors could become available as well.
